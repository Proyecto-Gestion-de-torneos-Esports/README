# Plataforma gestión de torneos eSports
Plataforma de gestión de esports encargada de la administración de torneos, usuarios, partidas, ranking global dentro de un sistema distribuidos basada en microservicios.

## Microservicios
- Microservicio de Usuarios
- Microservicio de Equipos
- Microservicio de Torneos
- Microservicio de Partidas
- Microservicio de Estadisticas
- Microservicio de Inventario de Premios
- Microservicio de Reglas del Torneo
- Microservicio de Recompensas
- Microservicio de Notificaciones
- Microservicio de Ranking
- Microservicio de Auditoria
- Microservicio de Juego

## Tecnologías Utilizadas
- Base de datos MySQL
- Lenguaje de programación Java versión 21
- Framework Spring Boot
- IDE IntelliJ IDEA
- Data Modeler
